# 🌿 Greenden – Flower Pot Platform 🪴

Greenden is a modern and responsive front-end website that showcases a variety of natural and artificial flower pots.  
This project focuses on clean UI design, responsive layouts, and smooth navigation.

---

## 🪴 About Greenden

Greenden is built for plant lovers who want to explore stylish and decorative flower pots for homes and offices.  
The platform highlights both natural and artificial plant collections with a simple and attractive interface.

---

## 🚀 Features

- 🌿 Modern and clean homepage
- 🪴 Dedicated products page
- 📩 Contact page
- 📱 Fully responsive design
- 🎨 Styled using Tailwind CSS
- 🖼️ Organized images folder

---

## 🛠️ Technologies Used

- Git & GitHub
